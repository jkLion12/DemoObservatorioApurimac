Registrar noticia
Recomendaciones
  • Título breve y orientado a una sola idea.
  • Fecha y lugar consistentes con la publicación.
  • Portada clara y útil para web.
Registrar infografía
Enfoque visual
  • Usar títulos legibles.
  • Priorizar un solo mensaje clave.
  • Preparar salida para portal y redes.
Registrar video
Buenas prácticas
  • Preferir videos breves y claros.
  • Describir propósito y audiencia.
  • Preparar miniatura y nombre consistentes.
Registrar usuario
Control de acceso
  • Usar correos institucionales.
  • Asignar estados de cuenta claros.
  • Documentar altas y revisiones.
Registrar entidad aliada
Relación institucional
  • Definir claramente el rol del aliado.
  • Usar descripción breve y formal.
  • Agregar identidad visual cuando exista.
